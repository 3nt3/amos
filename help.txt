Here are all commands:
```!kick @name```kicks a user from the server **requires permissions**
```!ban @name```bans a user from the server **requires permissions**
```!getLog```get the log file **admin only**
```!clearLog```clears the log file **admin only**
```!delete [amount]```deletes [amount] messages (min: 1; max: 100; messages being older than 2 weeks will not be deleted)
```!play [url]```plays a song in the voice chat BETA
```!stop```stops the current song
```!code```Link to the git-repository
```!help```lists all commands
